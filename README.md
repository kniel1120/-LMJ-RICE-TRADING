# LMJ Rice Retailing POS

This project is a Point of Sale (POS) system for LMJ, a rice retailer business. The system manages suppliers, inventory, customers, cost, and retail price. It is designed to be simple, user-friendly, and suitable for small business operations.

## Features
- Supplier management
- Inventory tracking
- Customer management
- Cost and retail price management
- Sales processing

## Tech Stack
- Python (backend)
- SQLite (database)
- Tkinter (GUI)

## Getting Started
1. Ensure you have Python 3.8+ installed.
2. Run `python main.py` to start the application.

## Project Structure
- main.py: Application entry point
- models/: Data models for suppliers, inventory, customers, etc.
- views/: GUI components
- controllers/: Business logic
- database/: Database initialization and helpers
